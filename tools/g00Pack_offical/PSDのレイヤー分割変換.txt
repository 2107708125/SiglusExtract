PSDのレイヤー分割変換

フォトショップのＰＳＤ画像をレイヤー毎に変換して、それぞれのレイヤー名でG00画像が保存されます。
特定のレイヤー名（レイヤーコマンド）がある場合に処理されます。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■レイヤーコマンド

	◆ #CONVERTS 分割変換コマンド

		何も絵の描かれていない空のレイヤーを作り、そのレイヤーの名前を #CONVERTS にして下さい。
		※このレイヤーは切り出しの対象になりません。

		レイヤー名がそのままファイル名になります。
		※各調子[.g00]は自動的に付くので必要ありません。

	◆ // コメントコマンド

		レイヤー名の頭に // があるレイヤーは、無視します。
		※このレイヤーは切り出しの対象になりません。

		例. 下地 → //下地

■注意点

	G00pack.exeの日付を比較して更新されていたら選択状態にする機能（再検索ボタン）が使えません。
	その為、レイヤー毎にコンバートしなければならないＰＳＤ画像の更新は自己管理でお願いします。

	※「ＰＳＤ画像のファイル名」を「変換されるレイヤー名」のどれかと同じ名前にしておけば、
	　再検索ボタンが使えるようになります。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ファイル名リスト

	変換後のファイル名を列記したテキストを用意し、そのファイル名で保存させる事ができます。

	≪作成手順≫

		１．単なるテキストファイルです。新規テキストファイルで作成して下さい。

		２．ファイル名を画像と同じ名前にし、ファイルの拡張子を [.fnl] にして下さい。

			例）画像の名前が BustUp.psd の場合、ファイル名リストのファイル名前は BustUp.fnl です。

		３．リストを列記して下さい。（記述例は、FileNameListサンプル.fnl 参照）

	≪ファイル名に使用できる文字≫

		a〜z, A〜Z, 0〜9, _, （※ファイル名の先頭に「数字」は使えません。）

	≪リスト内のコメント記号≫

		;  //  /* 〜 */ が利用できます。

	≪補足説明≫

		上のレイヤーから順にリストの名前が割り当てられます。
		※途中、コメントレイヤーなど、変換の対象にならないレイヤーには名前は割り当てられません。

		リストの数がレイヤー数より少ない場合、足りなくなったレイヤー以後は、
		レイヤー名がそのままファイル名になります。
