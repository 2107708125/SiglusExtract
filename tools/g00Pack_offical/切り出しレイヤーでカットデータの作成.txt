切り出しレイヤーでカットデータの作成

フォトショップのＰＳＤ画像のレイヤーを利用して「切り出しデータ」を作成する事が出来ます。
特定のレイヤー名（レイヤーコマンド）がある場合に処理されます。

■「切り出し」とは

	「切り出し」とは、画像コンバート時に、複数の矩形に切り出す事です。
	通常コンバートの場合は、単純に１枚絵のデータが作成されますが、
	切り出しコンバートの場合は、切り出された複数のデータが連結しているデータとして作成されます。
	例えば、ロールプレイングゲームで使用するようなチビキャラクタの歩きパターンをパターンごとに
	切り出すような感じだと考えて下さい。

	「切り出しデータ」は、オブジェクトシステム用に特化した形式です。
	旧来のバンクシステムでも画像を問題なく読み込む事はできますが、切り出しておく意味がありません。

	≪注意点≫
		レイヤー切り出しは、全ての切り出しを「レイヤーのサイズ」で切り出します。
		※サイズの違う切り出しをする場合は、≪切り出しテキスト≫ の方法でしかできません。

	≪切り出しテキスト≫
		切り出す矩形サイズをテキストで指定して切り出す方法があります。
		どちらが適切な方法なのかは、画像の取り扱い方によります。
		「切り出しテキストでカットデータの作成.txt」もご一読ください。
		※１つの画像に対して「切り出しレイヤー」と「切り出しテキスト」を併用する事はできません。

■レイヤーコマンド

	◆ #CUT 切り出しコマンド

		何も絵の描かれていない空のレイヤーを作り、そのレイヤーの名前を #CUT にして下さい。
		※このレイヤーは切り出しの対象になりません。

		上のレイヤーから順に切り出しされます。

	◆ // コメントコマンド

		レイヤー名の頭に // があるレイヤーは、無視します。
		※このレイヤーは切り出しの対象になりません。

		例. 下地 → //下地

	◆ #C(x)(y) 中心指定コマンド

		何も絵の描かれていない空のレイヤーを作り、そのレイヤーの名前を #C(x)(y) にして下さい。
		※このレイヤーは切り出しの対象になりません。

		上のレイヤーから順に切り出されるので、中心を設定したいレイヤーの前に空レイヤーを作成して下さい。
		一度、中心を設定すると、以後の切り出しに適用され続けます。
		また、途中で何度でも変更可能です。

		◆簡易指定 ： #C(x)(y)

			≪ (x) パラメータ≫
				0 = 切り出し座標の左端
				1 = 切り出し座標の中間
				2 = 切り出し座標の右端

			≪ (y) パラメータ≫
				 0 = 切り出し座標の上端
				 1 = 切り出し座標の中間
				 2 = 切り出し座標の下端

			≪例≫
				#C00 = 左上(0,0)
				#C11 = 中心
				#C22 = 右下
				#C12 = Ｘ軸中心、Ｙ軸下端

		◆座標指定 ： #C

			そのレイヤーに１ピクセルの点を描画して下さい。（何色でもいいです。）
			その場所が中心になります。
			※２点以上描画した場合、先に見つかった方を中心と考えます。
